# RITU 2025

The official website for RITU 2025, the annual technical festival of the Govt. Rajiv Gandhi Institute of Technology,
Kottayam.

## Getting Started

Clone the repository and navigate to the project directory:
```bash
git clone https://github.com/fbn776/RITU-2025.git
cd RITU-2025
```

Install the dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```
